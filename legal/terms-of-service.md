## 2. Description of Service

Flash is a mobile application for climbers that enables users to:
- Photograph climbing walls and map holds onto wall images using ML-assisted detection
- Create and share bouldering problems ("routes") composed from mapped holds
- Track personal climbing progress through a tick system
- Discover public walls and problems created by other users
- Collaborate with others through groups

The Service requires an internet connection for data synchronization and account features. Some read-only features function offline using locally cached data.
